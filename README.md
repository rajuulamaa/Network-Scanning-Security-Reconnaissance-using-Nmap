# Network Scanning & Enumeration using Nmap

## Overview
Performed advanced network reconnaissance and enumeration using Nmap against public learning targets. Conducted multiple scan techniques to identify open ports, exposed services, firewall behavior, service versions, and operating system fingerprints.

## Environment
- Nmap 7.98
- macOS Terminal
- Zenmap GUI

## Targets Scanned
- target.com
- macys.com
- joespizzanyc.com
- bluestockings.com

## Scan Types Performed

### Host Discovery
